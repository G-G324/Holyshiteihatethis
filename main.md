# Go Figure More - Project Setup


This is the homepage build for **Go Figure More**, a modern streetwear eCommerce project.


## Structure
- `public/index.html`: The entry HTML file.
- `src/main.jsx`: React entry point.
- `src/App.jsx`: Homepage component.
- `src/styles.css`: Custom styles.
- `Main.md`: Documentation file (this file).


## Running the Project
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`
